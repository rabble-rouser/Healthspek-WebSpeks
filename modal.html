Close Modal | Details

Clicking/tapping Close Modal will close the modal.

----
HTML
----

<!--Our anchor has the data-hspek attribute set to `close`, which accepts an optional `href` attribute, but we are not sending one.-->

<a class="button" data-hspek="close">Close Modal</a>


--
JS
--

/* The `hspek.close` method accepts one optional parameter: the `href` to redirect to. */ hspek.close();

Close Modal & Redirect | Details

Clicking/tapping Close Modal & Redirect will close the modal and redirect to page1.html.

----
HTML
----

<!--Our anchor has the data-hspek attribute set to `close`, which accepts an optional `href` attribute, set to redirect to our `page1.html`.-->

<a class="button" href="http://rabble-rouser.github.io/Healthspek-WebSpeks/modal.html" data-hspek="close">Close Modal & Redirect</a>


--
JS
--

/* The `hspek.close` method accepts one optional parameter: the `href` to redirect to. */ hspek.close('modal.html');
