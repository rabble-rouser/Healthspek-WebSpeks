Normal Link Details

Clicking Normal Link will open a modal and load its contents in the current view.

<!--This anchor does not have data-spek, so it will not be intercepted.-->

<a class="button" href="page1.html">Normal Link</a>
Open Full-Screen Modal Details

Clicking Open Full-Screen Modal will open a modal, and load it's contents.

----
HTML
----

<!--This anchor has the data-hspek attribute set to `open`, which accepts the href attribute.-->

<a class="button" href="modal.html" data-hspek="open">Open Full-Screen Modal</a>


--
JS
--

/* The `hspek.open` method accepts two parameters, the first being the `href`, and the second being web view type `open`, which designates a full-screen modal. */

hspek.open('modal.html', 'open');
Open Regular Modal Details

Clicking Open Regular Modal will open a modal and load its contents.

----
HTML
----

<!--Our anchor has the data-hspek attribute set to `modal`, which accepts the href attribute.-->

<a class="button" href="modal.html" data-hspek="modal">Open Regular Modal</a>


--
JS
--

/* The `hspek.open` method accepts two parameters, the first being the `href`, and the second being web view type `modal`, which designates a regular modal. */

hspek.open('modal.html', 'modal');
